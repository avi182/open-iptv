# Contributing to OpenIPTV

Thanks for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/openiptv.git`
3. Install dependencies: `npm install`
4. Start the dev server: `npm run dev`
5. Create a branch for your change: `git checkout -b my-feature`

## Development

- Client runs at `http://localhost:4200`, API at `http://localhost:4201`
- Vite proxies `/api` requests to the backend automatically
- Run `npm run build` to verify the production build works

## Submitting Changes

1. Keep PRs focused — one feature or fix per PR
2. Make sure the project builds without errors
3. Write a clear PR description explaining what changed and why

## Reporting Issues

Open a [GitHub issue](https://github.com/avi182/openiptv/issues) with:
- Steps to reproduce
- Expected vs actual behavior
- Browser/OS if relevant
